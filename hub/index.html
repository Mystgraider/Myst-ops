myst-ops // control panel
Live Tools
Dashboard
live
One UI to trigger and monitor bogart, OpenOSINT, Argo, and security-analyst - no need to dig through GitHub's mobile Actions UI.
ThreatScanner
live
Browser-based recon: headers, CORS, SQLi/XSS indicators, subdomain enum, CSRF PoC, brute force, WebSocket fuzz, and static code scan.
GHOST CRM
live
Case management for OSINT/bug-bounty findings - people, businesses, locations, entity networks. Login required.
GitHub Actions (also usable directly from the Dashboard above)
Bogart Recon Scan
Actions
Subdomain enum + active vuln scan (nuclei + afrog) against a target you specify. Downloads a report artifact.
Security Analyst Scan
Actions
Static source-code scan (regex + AST) across this repo - hardcoded secrets, weak crypto, injection patterns.
OpenOSINT Investigation
Actions
Passive OSINT lookups - email, username, DNS, whois. Pick a tool and target from the run dialog.
Argo Static Audit
Actions
LLM-native code audit. Start with the free "mock" runner before spending anything on a real "headless" run.
Actions links open the run dialog on GitHub - you'll need to be signed in with write access to actually trigger a run (this page has no way to trigger anything on its own, on purpose - no token lives here). Source: github.com/Mystgraider/Myst-ops
